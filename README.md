# Ujed-Api

1. Clonar proyecto

2. Instalar dependencias
```
npm i
```

3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```

4. Levantar la base de datos
```
docker-compose up
```

5. Ejecutar en modo desarrollo
```
npm run start:dev
```
